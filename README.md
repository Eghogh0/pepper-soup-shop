# The Pepper Soup Shop

A premium, interactive landing page for a Nigerian food company built with Next.js 15, TypeScript, TailwindCSS, and cutting-edge animations.

## Tech Stack
- **Next.js 15 (App Router)**
- **TypeScript**
- **TailwindCSS**
- **Framer Motion**
- **GSAP ScrollTrigger**
- **Lenis Smooth Scroll**
